## üîÅ What Is a Pull Request?

A **pull request** is a way for someone to suggest changes to your code. Think of it like **tracked changes** in a Word document ‚Äî but for code.

In this demo:
- Your instructor has submitted a PR to give feedback
- You get to decide whether to accept (merge) it
- You can also respond, ask questions, or make more changes
